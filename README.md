# Baskin robbins clone

React18 + TypeScript + Vite

Pc / Tablet / Mobile

> 해당 사이트는 2024.06.12을 기준으로 제작되었습니다.

|Menu|- 이달의 맛<br />- 아이스크림<br />- 프리팩<br />- 아이스크림 케이크<br />- 디저트<br />- 음료<br />- 커피|

|BR Story|- 이달의 맛 히스토리<br />- Be Better|
